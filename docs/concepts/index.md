# Knowledge Graph Concepts

A knowledge graph represents information as a network of entities and relationships.

## Core model

```txt
(Entity)-[Relationship]->(Entity)
```

Example:

```txt
(Neuro Graph)-[DOCUMENTS]->(Knowledge Graphs)
```

## Key parts

- **Entities** are nodes in the graph.
- **Relationships** are edges connecting entities.
- **Properties** describe entities or relationships.
- **Sources** explain where facts came from.
- **Queries** help discover connected information.
