# Entities

Entities are the main objects in a knowledge graph. They represent real or abstract things.

## Common entity types

| Entity type | Example |
| --- | --- |
| Person | Ada Lovelace |
| Organization | OpenAI |
| Project | Neuro Graph |
| Document | API Guide |
| Concept | Knowledge Graph |
| Dataset | Research Corpus |

## Entity shape

```json
{
  "id": "concept:knowledge-graph",
  "type": "Concept",
  "name": "Knowledge Graph",
  "description": "A network of entities and relationships."
}
```
