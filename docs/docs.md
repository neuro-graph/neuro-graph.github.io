# Neuro Graph Documentation

Neuro Graph is an open-source documentation project for knowledge graph systems. It explains how to represent entities, relationships, sources, and queries in a way that is easy for developers and researchers to understand.

## What you can document here

- Entity types such as `Person`, `Organization`, `Document`, `Dataset`, and `Concept`
- Relationship types such as `KNOWS`, `WORKS_FOR`, `MENTIONS`, `DEPENDS_ON`, and `RELATED_TO`
- Query patterns for graph traversal and discovery
- API examples for creating, reading, and exploring graph data
- Community notes and contribution guidelines

## Why Neuro Graph?

Knowledge graphs make information easier to connect. Neuro Graph focuses on the documentation layer: clear concepts, reusable examples, and developer-friendly explanations.
